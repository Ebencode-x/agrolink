# Security Policy

## Supported Versions
Currently, Agrolink is in active development. Only the main branch is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability
I take security seriously. If you find a bug or a vulnerability, please do NOT create a public issue. Instead, report it privately to:

* **Contact:** Ebencode-x
* **Response Time:** Within 48-72 hours.

Thank you for helping keep Agrolink secure!
